# RESCENE Fan Site

르센느(RESCENE) 공식 팬사이트 — Next.js 기반 K-pop 팬 커뮤니티 프로젝트

> **RESCENE** = RE(다시) + SCENE(장면)  
> 향기로 다시 장면을 떠올린다는 프루스트 효과에서 이름을 따온 5인조 걸그룹

## Features

- 홈 페이지 — 히어로, 멤버 프리뷰, 최신 앨범, 뉴스, 스케줄, 공식 링크
- 멤버 프로필 — 5인 개별 소개
- 음악 — 디스코그래피 전체 목록
- 스케줄 — 활동 타임라인
- 커뮤니티 — 뉴스 & 팬 커뮤니티
- 갤러리 — 포토 갤러리

## Getting Started

```bash
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev
```

개발 서버 실행 후 [http://localhost:3000](http://localhost:3000) 접속

### Commands

```bash
npm run dev      # 개발 서버 (localhost:3000)
npm run build    # 프로덕션 빌드
npm start        # 프로덕션 서버 실행
npm run lint     # 린트 검사
```
